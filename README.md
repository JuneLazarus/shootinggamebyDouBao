# 彩铅线稿 FPS (PencilLineFPS)

零依赖单文件网页第一人称射击游戏，手绘彩铅美术风格，本地双击打开即玩，数据结构与算法B课程大作业项目。

## 项目背景
本项目是一个基于 WebGL 技术实现的浏览器端第一人称射击游戏，旨在通过游戏开发实践，应用计算机图形学、物理模拟、碰撞检测等计算机基础知识，同时探索纯前端零依赖单文件应用的极致轻量化实现方案。项目采用原创彩铅手绘美术风格，所有资源程序化生成，无需任何外部依赖即可在本地运行。

## 核心知识点应用
本项目深度应用了以下课程相关知识点：
1. **AABB 轴对齐包围盒碰撞检测算法**：实现玩家、敌人、子弹与地形障碍物的碰撞判定，分离水平/垂直轴独立检测，支持自动台阶爬升、落地判定，碰撞精度容错5cm避免浮点误差卡脚
2. **三维空间向量数学**：所有移动、视角控制、子弹飞行、后坐力计算均基于 Three.js 三维向量库实现，包含向量归一化、点乘叉乘、线性插值等运算
3. **程序化生成算法**：
   - 随机地形生成：基于碰撞预检测避免地形重叠，高台自动生成垫脚台阶保证可达性
   - 程序化纹理生成：Canvas 2D 实时生成纸张纤维肌理、噪点颗粒效果，无需外部图片资源
4. **简单状态机与有限状态管理**：游戏状态（开始/游玩/暂停/结算）、敌人状态、玩家动作状态（站立/移动/疾跑/跳跃/瞄准/换弹）均采用状态机模式管理
5. **Web Audio API 音频合成**：所有射击、换弹、跳跃、受击、环境音效均通过振荡器+噪声生成器实时合成，零外部音频文件
6. **Canvas2D 后处理管线**：手写 RGB 通道偏移色差、纸张纹理叠加、动态噪点效果，替代第三方后处理库解决本地打开跨域问题

## 运行指南
### 环境要求
- 任意现代浏览器（Chrome 90+ / Edge 90+ / Firefox 88+ 均支持）
- 无需安装任何依赖、无需服务器环境、无需联网
- 支持 Windows/macOS/Linux 全平台

### 运行步骤
1. 直接下载项目中的 `shoot.html` 文件
2. 双击文件使用浏览器打开即可开始游戏
3. （可选）将自定义背景音乐命名为 `bgm.mp3` 放在同一目录下，可自动循环播放自定义音乐

### 操作说明
| 按键 | 功能说明 |
| ---- | ---- |
| W/A/S/D | 前后左右移动 |
| Shift | 按住疾跑（移动速度提升 25%，消耗体力，瞄准时无法疾跑） |
| 空格 | 跳跃（消耗 15 点体力，体力耗尽时跳跃高度减半） |
| 鼠标移动 | 控制视角方向 |
| 鼠标左键 | 射击 |
| 鼠标右键 | 按住瞄准（视角收窄、移速降低、弹道散布极小，适合远距离精准射击） |
| R 键 | 换弹 |
| ESC 键 | 暂停游戏 / 解锁鼠标 |

### 游戏特色
- 极致轻量化：单 HTML 文件 ~60KB，可离线运行，U盘便携即开即玩
- 彩铅手绘风格：米黄纸张肌理、逐帧抖动铅笔轮廓、RGB 微色差模拟叠色效果
- 完整射击手感：动态腰射散布、机瞄过渡、后坐力反馈、爆头三倍伤害判定
- 随存活时间动态递增难度：敌人移速/血量/生成密度/伤害随游戏时长逐步提升
- 随机生成地图：80×80米地形每次重开随机生成，所有高台均有垫脚可达
- 敌人简单绕障AI：撞墙时自动尝试左右绕行/后退，减少卡墙概率
- 枪口火焰、墙面弹孔效果，丰富战斗反馈
- 零外部资源：所有模型、音效、纹理全部代码生成
